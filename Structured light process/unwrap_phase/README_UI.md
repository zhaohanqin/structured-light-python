# 相位解包裹程序 - 图形界面版

这是一个基于PySide6开发的相位解包裹程序图形界面，用于便捷地处理四步相移图像并生成解包裹相位结果。该程序适用于结构光3D扫描等应用。

## 功能特点

- 美观的现代化图形界面，使用浅色系配色
- 支持同时处理水平和垂直方向的相位解包裹
- 支持三种解包裹算法：质量引导法、多频率法和时序相位解包裹
- 自动识别文件夹中的图像，前四张用于水平方向，后四张用于垂直方向
- 实时显示处理进度和结果
- 支持保存和查看解包裹结果

## 相位解包裹方法

本程序实现了三种主要的相位解包裹算法：

### 1. 质量引导法（Quality-guided Unwrapping）

这是一种空间解包裹方法，通过相位质量图引导解包裹路径。算法从质量最高的点开始，按照质量降序进行广度优先搜索，计算相邻像素间的相位差异。这种方法适用于单频率相位图，对噪声和不连续区域有较好的鲁棒性。

### 2. 多频率法（Multi-frequency Unwrapping）

利用不同频率的包裹相位进行解包裹，从最低频率开始，逐步利用高频率相位提高分辨率。通过计算相邻频率之间的频率比和相位差异，实现从低频到高频的解包裹过程。多频率法能有效解决单频率相位图中的包裹歧义问题，提高解包裹的准确性。

### 3. 时序相位解包裹（Temporal Phase Unwrapping）

时序解包裹综合了多频率方法和质量加权策略，它不仅考虑不同频率的相位关系，还结合相位质量图进行加权融合。该算法计算高频相位的预测值与实际值之间的差异，确定相位偏移量，并根据质量图调整不同频率解包裹结果的权重。这种方法在处理复杂场景时能获得更平滑和准确的结果。

## 图形界面特性

- **多方向处理**：同时支持水平和垂直方向的相位解包裹
- **实时进度显示**：通过进度条和状态提示实时显示处理进度
- **方向选择**：可选择仅处理水平方向、仅处理垂直方向或同时处理两个方向
- **结果可视化**：自动显示处理结果，包括包裹相位和解包裹相位图像
- **快速预览**：点击结果缩略图可查看全尺寸图像

## 安装依赖

使用pip安装所需依赖：

```bash
pip install -r requirements_ui.txt
```

主要依赖包括：

- numpy
- opencv-python
- matplotlib
- PySide6

## 使用方法

### 启动程序

```bash
python unwrap_phase_ui.py
```

### 界面操作流程

1. **选择图像**：
   - 点击"选择图像文件夹"按钮自动加载文件夹中的图像
   - 或分别点击"选择水平方向图像"和"选择垂直方向图像"手动选择四步相移图像

2. **设置解包裹方向**：
   - 选择"仅水平方向"、"仅垂直方向"或"两个方向"

3. **选择解包裹算法**：
   - 从下拉菜单中选择所需的解包裹算法
   - 可选：质量引导法、多频率法、时序相位解包裹

4. **设置输出目录**：
   - 点击"选择..."按钮设置结果保存位置

5. **开始处理**：
   - 点击"开始处理"按钮启动解包裹过程
   - 处理完成后，结果会显示在界面下方的图像显示区域

6. **查看结果**：
   - 点击"查看结果文件夹"按钮打开输出目录
   - 程序会保存包裹相位图、解包裹相位图和相位质量图

## 输出结果

在输出目录中，程序会为水平和垂直方向分别创建子文件夹，每个子文件夹包含：

- `wrapped_phase.png`: 包裹相位可视化图像
- `unwrapped_phase.png`: 解包裹相位可视化图像
- `wrapped_phase.npy`: 包裹相位数据 (NumPy数组)
- `unwrapped_phase.npy`: 解包裹相位数据 (NumPy数组)
- `quality_map.npy`: 相位质量图数据 (NumPy数组)

## 线程处理

程序使用多线程处理相位解包裹，以避免界面卡顿：

- 所有耗时操作在单独的工作线程中执行
- 处理过程中界面保持响应
- 使用信号槽机制更新进度和显示结果
- 基于`unwrap_phase_modified.py`的多线程安全设计

## 注意事项

- 输入图像应为灰度图像
- 四张图像的尺寸应完全相同
- 相位偏移应按照0°, 90°, 180°, 270°的顺序排列
- 水平方向相位图来自垂直条纹图像，垂直方向相位图来自水平条纹图像
- 当使用"temporal"或"multi_freq"方法处理单频率数据时，程序会自动使用"quality_guided"方法代替
